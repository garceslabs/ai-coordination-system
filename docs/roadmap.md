# Roadmap

## Phase 1 — Core coordination model
- Define task, agent, routing decision, and execution result models.
- Implement deterministic routing rules.
- Add basic escalation triggers.

## Phase 2 — Reliability layer
- Add retries, fallbacks, confidence thresholds, and validation hooks.
- Capture structured logs for every decision.

## Phase 3 — Agentic workflow support
- Support multi-step plans.
- Add dependencies between tasks.
- Track task state transitions.

## Phase 4 — Human-in-the-loop review
- Add review queue abstraction.
- Define approval, rejection, and correction flows.

## Phase 5 — Production extensions
- API layer.
- Queue integration.
- Persistent database.
- Dashboard.
- Cost and latency metrics.
